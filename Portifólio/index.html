Seja bem-vindo(a) ao portifólio

Aqui você encontrará todos os projetos desenvolvidos por Isabela Oliveira Fernandes, durante o curso Informática para Internet no ano de 2025!

Projetos:

1º Projeto: Fenda do Biquíni

O projeto "Fenda do Biquíni" foi inspirado no seriado infantil Bob Esponja e tem como objetivo apresentar um pouco mais sobre os principais personagens do desenho. A proposta buscou estimular a criatividade e a curiosidade, ao mesmo tempo em que consolidou os aprendizados de layout e organização visual.

2º Projeto: Blog do Woody

O projeto Blog do Woody foi desenvolvido com o objetivo de explorar e compartilhar curiosidades sobre o personagem Woody, do universo Toy Story. O projeto permitiu trabalhar habilidades de design de páginas, organização de conteúdo.

3º Projeto: Estrutura do Bingo

Este projeto teve como proposta desenvolver uma interface de Bingo simples, com foco no aprendizado e aplicação prática de conceitos de posicionamento e layout. O projeto permitiu exercitar a organização de elementos em tela de forma estruturada promovendo uma melhor compreensão sobre o comportamento dos grids e sua importância na construção de interfaces bem alinhadas e visualmente agradáveis.

4º Projeto: Viagem com Sentido

'Viagem com Sentido' é um site desenvolvido com o objetivo de indicar destinos incríveis para quem busca experiências únicas ao viajar. O projeto foi especialmente voltado para o estudo e aplicação de técnicas de responsividade, priorizando a navegação fluida e otimizada em dispositivos móveis, permitindo uma imersão prática no universo do web design responsivo.

5º Projeto: Equipes da F1 2025

O projeto 'Equipes da F1 2025' é um site desenvolvido com o objetivo de dar uma breve informação sobre as equipes que compõem a Formula 1, no ano de 2025. O projeto foi especialmente voltado para aplicação e estudo dos estilos já predefinidos do bootstrap.
